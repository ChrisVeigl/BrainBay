This software application enables to upload firmware to Neurobit devices. It is also possible to read out information about the hardware and firmware from the device. Wireless (Bluetooth or IrDA) or USB communication with the equipment is used. The application works under Microsoft® Windows® operating systems: 10/8/7/Vista/XP/2000.

Parameters

  • Firmware file - filename of binary file of firmware version selected to upload.
  • Access key - a key string entitling you to a specific update. If a given firmware is made generally available, this field can remain empty.
  • Link type - type of link used for communication with Neurobit device.
  • Device id - Identifier of a Neurobit device. It can be:
    • general description of a communication module:
      • "Neurobit Optima*" - for Neurobit Optima+ and Neurobit Optima USB/BT,
      • "Serial Port Device" - for Neurobit Optima,
    • Bluetooth address or
    • USB module serial number.
    If you have only one turned on Neurobit device paired with your computer or connected to it via USB, you can use the mentioned generalized identifier in this field.

Buttons

  • Browse - opens a window enabling selection of a specific version firmware file,
  • Check - reads out hardware and firmware information from the device,
  • Upload - uploads a firmware to the device,
  • Help - displays this help for the loader application,
  • Exit - closes the loader application.

Device info

This information appears after connection to a device:

  • Hardware - device model and serial number,
  • Transceiver id - communication module identifier, i.e.:
    • Bluetooth address or
    • USB serial number,
  • Bootloader - name and version of bootloader program residing in the device,
  • Old firmware - name and version of previous firmware,
  • New firmware - name and version of just uploaded firmware.

Update procedure

Prerequisites

  • Neurobit Optima[+] Bluetooth devices:
    Bluetooth communication port (built-in into a computer or as a USB adapter) compatible with the protocol version 1.1 or newer is required. To install Bluetooth adapter follow Neurobit Optima Quick Start Guide.
    Neurobit Optima device should be set in wireless communication range (usually a few meters). Before the use a Bluetooth device should be paired with your computer. Pairing code for Neurobit Optima[+] is 0000. You can learn more about Bluetooth pairing in Windows help.

  • Neurobit Optima[+] USB devices:
    A cable with full size USB 2.0 male A connector on one end and micro B male connector on the other end (the same as for typical cell phone chargers) is required. Install driver software for USB chip by FTDI, as described in the device quick start guide. Connect the computer to the device and turn it on. The computer should detect the device and install the driver necessary for its USB module.

  • Neurobit Lite devices:
    Infrared (IR) communication port (built-in into a computer or in form of a USB adapter) compatible with IrDA standard v. 1.0 or newer is required. To install IrDA adapter follow an instruction delivered in its package.
    The Neurobit Lite device should be set with IR port oriented towards IR port of your computer, no more than 0.5m from it, without physical obstructions between both entities, preferably on the same axis. There should be no other active IrDA devices nearby.

Typical firmware update

  1. Download the latest firmware binary:

    • Neurobit Optima+ models and Neurobit Optima 2 BT / USB devices,
    • older Neurobit Optima 2 / 4 models,
    • Neurobit Lite.

  2. Run Winload program:

    • from disk or
    • via supported biofeedback applications, in Neurobit device settings window, with Device services button on the General tab.

  3. Click Browse to select downloaded firmware file and choose Link type used by your Neurobit device. (If necessary, write down the Device id.)

  4. Turn on Neurobit device (and wait for its beep signal).

  5. Click Upload button of the application. Information about the hardware and its firmware is displayed. The bar at the bottom of the window indicates firmware transmission progress. The upload usually takes less than a dozen seconds for Bluetooth and USB.

  6. After successful transmission the new firmware is automatically launched. It is signalled with a beep.
    (In Neurobit Lite the new firmware name and version will be displayed on device's screen.)

  7. You can now turn off the device and close the loader application.

Remarks

  • If there are several Neurobit Optima[+] devices paired with / connected to your computer, we recommend to turn off all of them but the one intended for firmware update.
    (Otherwise the updated device should be unambiguously identified with its transceiver identifier set in the Device id.)

  • If a firmware transmission failed for any reason (like loss of connection between the devices or heavy load of processor), try to upload the firmware again.

  • If you plan to use several Neurobit devices (having the same link type) with the same computer concurrently, you need to identify each device unambiguously with its transceiver identifier. To be able to do so follow the procedure at the beginning:

    1. Run the Neurobit loader application (directly or using the Device services button on the General tab of the device settings window).
    2. Set suitable Link type. In the Device id field leave (or enter) the general identifier "Neurobit Optima*".
    3. Turn on only one Neurobit device (it should be paired with your computer or connected to it via USB at first).
    4. Click the Check button.
    5. When an identifier appears in the Transceiver id field, write it on a label and stick it to the device.
    6. Turn off the device.
    7. Repeat points 3-6 for each other Neurobit device.

    From now on you can use these identifiers to specify, which device piece will be used in a given instance.

  • If Neurobit Lite configuration structure has changed in new firmware version, default configuration is set. It is indicated with an appropriate message on device's display. In this case please make again startup settings, as described in the chapter "The basics of usage/Initial configuration" of your equipment instruction manual (especially System/Mains option, according to your country). You may also want to set other configuration options then, depending on your preferences.
    For most updates the device's configuration is preserved.

  • You can also consult the chapter "Firmware update" of device instruction manual.
